Leadra App Update Summary

This report summarizes the latest implemented improvements across the Leadra app, focused on easier unit discovery, smoother unit creation, cleaner client documents, and a more polished review experience.

Implemented Changes

Global Unit Search

The Units page now supports searching and filtering across all destinations and projects immediately, without requiring users to select a destination first.

Improved Browsing Flow

Destination and project cards remain available as quick browsing shortcuts, while global search gives users a faster way to find units directly.

Owner Name Flexibility

Users can now reuse an owner name when creating a new unit. Duplicate prevention remains focused on owner phone numbers where it matters.

Cleaner Client PDF Names

Shared client PDFs now use shorter, cleaner filenames that keep the key unit information and remove unnecessary extra details.

Media Review Polish

The media review area has been improved for readability, especially in dark mode, making uploaded images and review information easier to see.

Mobile Layout Refinement

The create-unit review screen now behaves better on mobile, with actions positioned so they do not cover uploaded media.

Smoother Unit Creation

The create-unit flow has been strengthened so users receive clearer guidance if something cannot be saved successfully.

Live App Update

The latest version has been applied to the live Leadra app and checked for availability.

Client Benefit

  • Users can find units faster from a single place.
  • Data entry is less restrictive and better matches real usage.
  • Client-facing PDF files look cleaner and easier to share.
  • The create-unit experience is clearer and more reliable.
