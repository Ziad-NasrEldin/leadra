# Leadra Progress Report

## Completed Items

The following requested updates have been completed and are ready for review.

- Comma formatting was added for number fields to make millions, prices, and month values easier to read.
- The Special page is visible to all users. Only admin and sub-admin users can edit Special units.
- Newly created units now appear faster in unit pages after creation.
- A smart paste field was added to create units faster by filling unit details from pasted information.
- Special units now include a button that generates ready-to-post copy for Facebook and social media.
- The thumbnail image frame and “confidential” text were removed from generated PDFs.
- Generated PDFs now show a maximum of two images per page in the image grid.
- Total amount is no longer adjustable by users.
- Installments are applied automatically on the first day of the month.
- A day value was added to the installment timetable to allow easier testing and control.
- The back button now moves one step back instead of jumping too far back.
- The light and dark theme toggle was adjusted to reduce screen flashing.
- Login behavior was improved so users can get into the app faster.
- Sales representatives cannot edit restricted unit controls or management-only actions.
- Paid amount was removed. Down payment is now treated as the paid amount and updates when installments are paid.
- Sales users cannot mark units as sold, on hold, or change unit status. They can still generate unit PDFs.
- The manual “mark as paid” feature was removed from installments.
- The app no longer signs users in automatically after they log out.

## Status

All listed items have been completed.
